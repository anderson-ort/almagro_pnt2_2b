# Plan: Aplicar estilo Clean al componente Catalogo

## Objetivo
Adaptar el archivo `Catalogo.css` para que siga los principios de diseño Clean (minimalismo, espaciado generoso, tipografía legible, paleta de colores limitada).

## Análisis del componente

El componente `Catalogo.vue` usa las siguientes clases CSS:
- `.stock-bajo` - para stock bajo (< 3)
- `button:disabled` - estado deshabilitado
- `.reservas-info` - contenedor de info de reservas
- `.max-alert` - alerta de máximo alcanzado
- `.filtro-input` - input de búsqueda
- `.sin-resultados` - mensaje sin resultados

### 2. Actualizar estilos del componente

| Clase | Cambio |
|-------|--------|
| `.stock-bajo` | Usar `--color-warning` en lugar de #e67e22 |
| `button:disabled` | Mantener opacidad 0.5, añadir cursor y transición |
| `.reservas-info` | Fondo con `--color-neutral-100`, padding y border-radius según tokens |
| `.max-alert` | Usar `--color-danger`, peso 600 |
| `.filtro-input` | Estilo clean con border, padding, focus states |
| `.sin-resultados` | Fondo warning suave, border-radius, typography |

### 3. Añadir estilos faltantes para elementos del template

- `ul` - lista sin estilos, spacing entre items
- `li` - padding, border inferior
- `button` - estilo primario Clean (background primary, padding, border-radius)
- `h2` - heading con font-display
