# Consigna — Simulación de carga de datos desde API

Desarrollar una página web simple utilizando **HTML y JavaScript** que permita simular la carga de información desde una API.

La API a consumir será:

```text
https://rickandmortyapi.com/api/character
```

---

## Requerimientos

1. Crear un botón en el HTML con el texto: 'Carga de info'
2. Al hacer clic en el botón, realizar la petición a la API.
3. Simular una espera de **2 segundos** antes de mostrar la información.
4. Mostrar en pantalla al menos **5 elementos** obtenidos de la API:

   * título
   * imagen

5. El botón debe permitir ejecutar la petición **solo una vez**, aunque el usuario haga múltiples clics.
